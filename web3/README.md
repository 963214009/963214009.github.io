# sgk
仿findmima的社工库代码，基于elasticsearch和PHP构建

整个架构用到的东西是</br>

1.filebeat</br>
2.logstash</br>
3.ealsticsearch</br>
5.php</br>
6.nginx</br>


#优点

1.基于elasticsearch，在数据的处理上要优于sphinx</br>
2.数据清洗，格式化和备份都比solr和sphinx要简单</br>
3.速度快啊</br>

#缺点

1.前台查询语句还不够好</br>
2.界面直接用的findmima的，有些样式还不够完美</br>
3.希望UI/前端大牛能bugfix</br>

#后续

1.可以开源清洗规则给大家</br>
